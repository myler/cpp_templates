# C++ Templates (2nd Edition)

### Study environment
```
OS: Ubuntu 18.04.2 LTS
Compiler: gcc version 7.4.0 (Ubuntu 7.4.0-1ubuntu1~18.04.1)
Meta-build tool: cmake version 3.4.3
build tool: GNU Make 3.81
```

### Build all examples
```
mkdir build
cd build
cmake ..
```

### Run all examples
```
ctest --verbose
```
